link 标签是干嘛的？

link 标签是定义文档与外部资源的关系。link 标签是最常见的用途是链接样式表。通过预处理提升渲染速度 在我们对中大型项目进行性能优化时，往往会对资源做减法（gzip 压缩，缓存等）或除法（按需打包，按需加载），可是如果能想到 link 标签的 rel 属性值来进行预加载，也能加快页面的渲染速度。
